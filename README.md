# Connected Worlds: Solving Real-World Problems with Graph Theory

## The Project
Graphs are the backbone of modern infrastructure, from social networks to emergency routing. This project implements fundamental graph algorithms to solve logistics and connectivity challenges.

## Solutions Provided
* **Social Connectivity**: Implementing BFS and DFS to suggest "friends" or explore network depth.
* **Dependency Mapping**: Using Topological Sorting to manage complex task schedules.
* **Smart Routing**: Applying Dijkstra’s algorithm for emergency vehicle pathfinding and Prim’s algorithm for cost-efficient network cabling.

## Technical Implementation
* **Data Structures**: Compared Adjacency Matrices for dense graphs vs. Adjacency Lists for memory-efficient sparse networks.
* **Analysis**: Each algorithm includes a performance breakdown, capturing the trade-offs between execution speed and memory overhead.
